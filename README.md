Inventory Audit & Supplier Performance Dashboard

This interactive dashboard developed using Microsoft Power BI provides detailed insights into inventory audits, supplier distribution, stock availability, and material costs across multiple store locations.

The dashboard presents key operational KPIs including Audit Total Items, Reorder Level, Inventory Value, Material Cost, and Stock Quantity. These high-level metrics allow managers to quickly assess the overall inventory health and financial impact of stock management.

Key insights provided in the dashboard include:

Inventory Value by Component to analyze the contribution of different components such as steel rods, bearings, gear shafts, and conveyor belts to the overall inventory value.
Supplier Distribution by Store and Category to understand how suppliers are associated with different store locations and product categories.
Material Cost Analysis showing cost variations across categories such as Mechanical, Electrical, Packaging, and Raw Materials.
Store-wise Supplier Performance Table summarizing supplier contributions across multiple store locations including Bangalore, Chennai, Coimbatore, Madurai, and Trichy.
Monthly Stock Quantity Trends to track inventory movement and identify seasonal patterns in stock levels.

Interactive filters including Category, Supplier Name, Store Name, and Purchase Order Date allow users to dynamically analyze the inventory data and explore specific operational scenarios.

This dashboard helps organizations to:

Monitor inventory audit performance
Track supplier distribution across stores
Analyze material cost by category
Identify stock quantity trends over time
Improve supply chain and inventory management decisions

Overall, this project demonstrates how Power BI can be used to build advanced inventory monitoring and supplier analytics dashboards for data-driven supply chain management.
